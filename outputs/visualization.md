# Visualizing each patient data - individual level

For each clinical trial create a folder that contains one figure for each patient.
The figure must include the patient PK, and PD data over time. Time of 0, is the time were patients received their first dose.
For each plot make sure the dose time is shown as vertical line. The dose amount could be shown in the legend or title of the figure.
Write the visualization code as python script and saved it under the scripts folder
